# Criando o Meu Portifolio

## Descrição

   Criar o meu portifolio, utilizando React e TypeScript, para exibir minhas habilidades e projetos.
   A ideia é que meu portifolio seja um site responsivo, que exiba minhas habilidades e projetos de forma clara e simples.
   O layout do site deve se basear no design do arquivo videoframe_8919.png, com uma navegação fácil e uma apresentação clara das informações.

## Requisitos Técnicos 
   - O projeto deve utilizar React e TypeScript.
   - Utilizar boas práticas de codificação, como nomeação consistente, indentação e comentários.
   - Utilizar componentização para organizar o código e facilitar a manutenção.
   - Pode deletar todos os arquivos do projeto, exceto os que são necessários para a execução do projeto.
   - Utilizar React Icons para adicionar ícones ao site.
   - Utilizar React Router DOM para criar rotas no site.
   - Quero um campo que terá a minha foto de perfil, de forma aredondada como na imagem do design.
   - O campo de foto de perfil terá ao redor dele uma animação de icons das minhas redes sociais, como LinkedIn, GitHub, Twitter, entre outros.
   - eles serão links que irá direcionar para as minhas redes sociais.
   - Esses icones terá uma animação que ficará rodando sobre minha foto do perfil (veja o exemplo na imagem videoframe_8919.png)
   - Utiliza o Tailwind CSS os elementos do site.
   - Utilize cores suaves para o site, como na imagem do design.
   - Utiliza o Tailwind CSS para estilizar os elementos do site.
   - Utilizar tipagem fortemente, como TypeScript, para garantir a segurança do código.
   - Crie Paginas para Perfil Profissional, Principais Tecnologias, Experiências Profissionais, Formação, Habilidades Técnicas e Projetos.
   - Crie botões que irão direcionar para cada página.
   - Utilize o Tailwind CSS para estilizar os botões.
   - Utilize cores suaves para os botões, como na imagem do design.
   - Utilize animações para os botões, como na imagem do design.
   - Utilize o Tailwind CSS para estilizar os elementos da página.
   - Veja o exemplo na imagem videoframe_8919.png para entender como deve ser o layout do site.
   - Veja o video exemplo.mkv para entender como deve ser o layout do site.
